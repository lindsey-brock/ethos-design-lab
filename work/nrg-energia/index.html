Overview

A 20-year-old company with a
website that didn't show it.

NRG Energia is a leading renewable energy and roofing contractor in Emilia-Romagna with two decades of experience, certified asbestos remediation credentials, and a growing residential solar portfolio. Their existing website ran on a generic WordPress template ("Organized" by Automattic) — off-brand, bloated with unnecessary plugins, technically non-compliant, and poorly positioned for local search. We rebuilt it from the ground up in clean HTML/CSS/JS, cutting the CMS dependency entirely.

UX Audit · The Problem

Eight critical issues
we found and fixed.

Before writing a single line of code, we audited the existing site across UX, performance, compliance, and SEO. Here's what we found.

📱

Poor Mobile UX

The WordPress theme had basic mobile support — a hamburger menu and stacked layout — but the mobile experience was an afterthought. No touch-optimized CTAs, no mobile-specific hero layout, oversized logo lockup, and the language selector dropdown dominated the top of the screen on small devices.

Six Languages via Google Translate Plugin

Rather than a proper bilingual site, the old site used a WordPress Google Translate plugin offering Arabic, Chinese, French, Italian, Portuguese, Spanish, and English — machine-translated, SEO-damaging, and completely misaligned with the actual client base. Google penalises auto-translated pages. The redesign replaced this with a clean, hand-translated IT/EN toggle.

Full Responsive Rebuild

Rebuilt from scratch in semantic HTML/CSS with mobile-first breakpoints. Separate navigation patterns, touch-optimized carousels, and a distinct mobile hero layout optimized for thumb reach.

UX

GDPR Cookie Consent System

Implemented vanilla-cookieconsent with granular consent categories (necessary, analytics, marketing). Privacy policy and cookie policy pages written and published. No tracking fires before consent.

Compliance

Cloudinary Image CDN

All images served via Cloudinary with automatic WebP conversion, responsive srcsets, and lazy loading. Page weight reduced by ~95% on the homepage. Core Web Vitals LCP improved dramatically.

Performance

SEO Foundation

Full meta tag implementation, semantic heading hierarchy, structured data markup, Open Graph tags for social sharing, XML sitemap, and robots.txt. Site is now fully indexable and positioned for local search.

SEO

Contact Form with Attachments + Resend

Custom serverless contact form with file attachment support (PDF, JPG, PNG up to 10MB). Form submissions delivered directly to Anna Rosa's inbox via Resend with full sender details and reply-to configured.

Integration

WhatsApp Business Integration

WhatsApp floating button with pre-populated message context, styled to match the site's design system. Direct channel to the team without requiring visitors to fill a form.

Integration

Italian / English Bilingual Toggle

Full IT/EN language toggle built in vanilla JS with localStorage persistence. All content — navigation, headings, body copy, CTAs, form labels — switches instantly without a page reload.

Accessibility

Clear CTA Hierarchy & User Journey

Every page now leads with one primary CTA ("Richiedi preventivo") and a secondary path via WhatsApp, with services organized by clear hierarchy so visitors always know the next step — from the homepage hero through to contact.

Conversion

Simplified Site Architecture

Removed the WordPress sidebar, search widget, and "Recent Articles" clutter from every interior page. Replaced with one consistent, simple navbar across the entire site and refreshed content throughout.
